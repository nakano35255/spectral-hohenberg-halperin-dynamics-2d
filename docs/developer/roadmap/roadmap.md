# ロードマップ (Roadmap)

このページは、ソルバーの最終的な設計目標と、現在の実装をつなぐための開発ロードマップです。

`docs/wiki/equations.md` は、このコードが最終的に扱いたい方程式系を定義する「設計の旗」として扱います。一方で、現在のコードはまだ時間発展ソルバーの入口にあり、実装済みの機能と今後の予定を明確に分けて管理します。

## 現在の実装済み機能

現在のコードベースでは、時間発展そのものに入る前のスペクトル計算基盤が実装されています。

- 2次元周期領域
- MPI による y-slab 分割
- HeFFTe/FFTW による R2C/C2R 変換
- スペクトル空間の保存変数 `State`
- 物理空間バッファ `PhysicalStateBuffer`
- input script parser
- density 初期条件: `uniform`, `sine`, `gaussian`
- momentum 初期条件: `uniform`, `sine`
- snapshot measurement: physical / spectral
- rank 0 のみの通常ログ出力

この段階の目的は、分散 FFT、データ配置、初期条件、出力の土台を固めることです。

## 未実装の主要機能

以下は `docs/wiki/equations.md` に現れるものの、まだ実装されていない、または実装途中の機能です。

- 時間発展 RHS
- time integrator
- 熱力学モデルの評価
- 化学ポテンシャルと圧力の自動生成
- 拡散フラックス
- 粘性応力
- 確率的フラックスと応力ノイズ
- density-dependent transport
- 勾配自由エネルギーと変分微分
- nonlinear term と dealiasing
- 静止流体モード
- 非圧縮性モードと射影法
- restart 入出力

これらは一度に実装せず、検証可能な小さな段階に分けて進めます。

## v0.1: 線形・決定論的時間発展

最初の時間発展実装では、非線形項とノイズを入れず、スペクトル空間で閉じる線形モデルを実装します。

目標:

- 線形熱力学モデル
- 線形 RHS
- Euler integrator
- 単一 Fourier mode の解析解比較
- MPI rank 数を変えても同じ結果になることの確認

この段階では、数式 parser や自由エネルギー自動微分には踏み込みません。FFT の符号、波数、配列レイアウト、時間積分の更新経路を先に固定します。

## v0.2: 理想気体・決定論的 FHD

次に、自由エネルギーから熱力学量を評価する最初の物理モデルとして、等温理想気体混合物を実装します。

目標:

- 理想気体自由エネルギー
- 化学ポテンシャル `mu_k`
- 熱力学的圧力 `p`
- 定数の移動度行列 `L_kl`
- 決定論的拡散
- 定数粘性による運動量拡散

この段階でも、ノイズと強い非線形物性はまだ扱いません。

## v0.3: 非線形項と物理空間評価

移流項や密度依存する熱力学量を扱うため、物理空間での評価バッファと疑似スペクトル更新を整備します。

目標:

- `State` から物理空間場への変換
- 速度場 `v = j / rho`
- 移流項
- nonlinear flux/stress buffer
- 必要に応じた dealiasing 方針の確定

この段階で、`docs/wiki/equations.md` の一般的な決定論的 FHD 方程式に近づきます。

## v0.4: 確率項

決定論的な計算経路が安定した後に、揺動散逸定理に基づくノイズを追加します。

目標:

- 質量フラックスノイズ
- 応力ノイズ
- additive noise 近似
- seed 管理
- MPI-safe random number generation
- 平衡揺らぎの統計検証

最初は `L_kl`, `eta`, `zeta` を定数に制限し、状態依存ノイズは後の段階に回します。

## v0.5: 勾配自由エネルギーと相分離モデル

界面を含むモデルへ拡張します。

目標:

- 勾配自由エネルギー
- 変分微分による化学ポテンシャル
- Cahn-Hilliard / Model B
- Navier-Stokes-Korteweg
- Model H への準備

この段階では、圧力テンソルを直接実装するか、化学ポテンシャル由来の力として扱うかを、数値安定性と実装の見通しを含めて決めます。

## v1.0: 汎用 FHD ソルバー

v1.0 では、`docs/wiki/equations.md` に書かれた主要な方程式系を、入力スクリプトから実用的に扱えることを目標にします。

目標:

- 多成分混合流体
- 自由エネルギーベースの熱力学
- 拡散・粘性・ノイズの整合的な実装
- 非線形・確率的時間発展
- snapshot / restart / diagnostics
- 代表的な検証問題の examples

この段階でも、すべての物理モデルを網羅することより、方程式・実装・検証が一貫していることを優先します。
