# `length`

## 目的

2次元周期領域の物理サイズを指定します。

## 形式

```txt
length <Lx> <Ly>
```

## 例

```sh
length 64.0 64.0
length 6.283185307179586 6.283185307179586
```

## 引数

- `<Lx>`
  - 型: double
  - $x$ 方向の系の長さ
  - 正の値である必要があります。
- `<Ly>`
  - 型: double
  - $y$ 方向の系の長さ
  - 正の値である必要があります。

## デフォルト値

省略時は以下が使われます。

```txt
length 6.283185307179586 6.283185307179586
```

実装上は、どちらの方向も $2\pi$ に初期化されています。
ただし、入力スクリプトで明示的に指定することを推奨します。

## 詳細

`length` は、計算領域

```math
\Omega = [0,L_x) \times [0,L_y)
```

の物理サイズを指定します。
Fourier空間の波数は、この長さを用いて

```math
k_x = \frac{2\pi n_x}{L_x},
\qquad
k_y = \frac{2\pi n_y}{L_y}
```

として定義されます。

格子間隔は、`grid` と `length` から決まります。

## 制限・注意

- `<Lx>` と `<Ly>` は正の値である必要があります。
- `length` は物理的な長さを決めるため、同じ `grid` でも `length` を変えると波数や微分演算のスケールが変わります。
- `dealias` を有効にしても、物理領域の長さ $L_x, L_y$ は変わりません。
