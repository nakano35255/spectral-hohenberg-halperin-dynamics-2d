# `timestep`

## 目的

時間発展で用いる時間刻み幅 $\Delta t$ を指定します。

## 形式

```txt
timestep <dt>
```

## 例

```sh
timestep 0.005
timestep 0.001
```

## 引数

- `<dt>`
  - 型: double
  - 1ステップあたりの時間刻み幅
  - 正の値である必要があります。

## デフォルト値

省略時は以下が使われます。

```txt
timestep 0.005
```

ただし、入力スクリプトで明示的に指定することを推奨します。

## 詳細

`timestep` は、`run` で指定した各ステップにおいて、物理時刻をどれだけ進めるかを決めます。

ステップ番号を $n$、時間刻み幅を $\Delta t$ とすると、物理時刻は概念的に

```math
t_n = n \Delta t
```

として進みます。

複数回 `run` を実行した場合も、ステップ番号と物理時刻はリセットされず、通算で加算されます。

## 制限・注意

- `<dt>` は正の値である必要があります。
- `0` 以下の値を指定するとエラーになります。
- 安定に計算できる `timestep` は、格子数、拡散係数、粘性係数、移動度、非線形項、ノイズ強度などに依存します。
- 大きすぎる `timestep` は数値不安定性の原因になります。
