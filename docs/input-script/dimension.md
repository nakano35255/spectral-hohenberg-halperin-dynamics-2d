# `dimension`

## 目的

シミュレーションの空間次元を指定します。

現在の実装は2次元専用です。

## 形式

```txt
dimension <num>
```

## 例

```sh
dimension 2
```

## 引数

- `<num>`
  - 型: integer
  - 指定可能な値:
    - `2`

## デフォルト値

省略した場合も、ソルバー本体は2次元ソルバーとして動作します。
ただし、入力スクリプトの意図を明確にするため、明示的に `dimension 2` を指定することを推奨します。

## 詳細

`dimension` は、入力スクリプトが2次元系を対象としていることを明示するためのコマンドです。

本ソルバーでは、計算領域を

```math
\Omega = [0,L_x) \times [0,L_y)
```

とし、密度場、運動量密度場、およびスカラー場を2次元周期領域上で時間発展させます。

## 制限・注意

- 現在指定できる値は `2` のみです。
- `1` や `3` などを指定するとエラーになります。
- 実装上、このコマンドは次元の切り替えではなく、2次元計算であることの検査として機能します。
